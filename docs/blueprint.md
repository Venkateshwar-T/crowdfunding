# **App Name**: Flarestarter

## Core Features:

- Campaign Listing: Display a grid of crowdfunding campaigns with filtering and sorting options based on category, funding goal, verification status, and asset acceptance.
- Campaign Details: Show detailed information for a selected campaign, including creator info, funding progress, accepted FAssets, milestones, and a mock real-time price component. Allow upload of proof-of-progress by creators using FileUpload UI Component. Transaction History shall show dummy data.
- Campaign Creation: Provide a form for creating campaigns, including fields for title, description, category, funding goal, deadline, milestones, images, identity verification, and asset acceptance.
- User Dashboard: A dashboard with tabs for managing created campaigns, viewing contribution history, initiating identity verification, and managing smart accounts, including dummy buttons.
- Identity Verification Simulation: Simulate an FDC identity verification process, allowing users to upload a mock ID and view their verification status with dummy info and buttons.
- FTSO Price Tool: Provide the actual price using a live price feed. Provide means of deciding whether to use the price feed using AI (reasoning).

## Style Guidelines:

- Primary color: Royal blue (#4169E1) for trust and innovation.
- Background color: Very light blue (#F0F8FF) to provide a clean and airy backdrop.
- Accent color: Deep purple (#800080) to add a premium and sophisticated feel.
- Headline font: 'Space Grotesk' sans-serif for headlines; Body font: 'Inter' sans-serif for body text
- Code Font: 'Source Code Pro' to be used for any displayed code fragments.
- Use modern, minimalist icons that represent Flare blockchain concepts and campaign categories.
- Implement smooth card layouts with rounded edges and glassmorphism effects to create a clean and premium design. Use GlassContainer reusable component.
- Incorporate subtle animations throughout the UI using Framer Motion for a smooth and engaging user experience.
